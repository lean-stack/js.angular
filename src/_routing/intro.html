Einführung - Location

Das Routing in AngularJS basiert auf zwei Komponenten:

  • Location-Service: $location
  • Routing von View und Controller
Letzteres wird durch zusätzliche Komponenten gewährleistet; dazu ist entweder das ngRoute-Modul zu installieren oder Fremdkomponenten wie zum Beispiel der AngularUI Router.

$location-Service

Der $location-Service bietet eine programmier-freundliche API auf die URL-Leiste des Browsers. Er ermöglicht ein bequemes Auslesen und Verändern der Bestandsteile (Protokoll, Host, Port, Pfad, Hash, Suche). Dabei wird die URL vom Service überwacht. Um Nicht-HTML5-Browser zu unterstützen wird dabei die Url verändert ausgewertet und jeder Link innerhalb einer Anwendung auch entsprechend abgefangen.
