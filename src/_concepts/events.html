Das Event-System

AngularJS definiert ein eigenes Event-System. Zugang erhalten wir über die Scope-API. Die wichtigsten Methoden:

  • $on Handler registrieren
  • $emit Event auslösen
  • $broadcast Event auslösen
Ein $emit verteilt das Event an die Kinde-Scopes, der $broadcast and die Parent-Scopes. Bei nicht in einer Beziehung stehenden Scopes kann der $rootScope genutzt werden. Link: Stackoverflow
